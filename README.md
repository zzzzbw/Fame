* 基于`node` `java` `spring-boot` `vue` `nuxt` 开发的博客系统
* 支持传统手动部署和`docker`部署
* 功能精简但齐全，界面简洁却美观，满足个人博客的日常使用要求
* 适合当做`'Javaer'`,`'Vuer'`的练手学习项目，也适合`'跨界开发'`做新技术涉猎的参考
* 项目会持续更新，如果有不完善的地方，欢迎指出

> 演示站点: [https://zzzzbw.cn](https://zzzzbw.cn)
